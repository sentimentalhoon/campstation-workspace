# 기능 명세서

> CampStation의 전체 기능 목록 및 우선순위

## 🎯 핵심 가치 제안

**CampStation**은 모바일 전용 캠핑장 예약 플랫폼으로, 사용자가 손쉽게 캠핑장을 검색하고 예약할 수 있는 서비스입니다.

## 📱 주요 기능 목록

### 1. 인증 및 사용자 관리 (Authentication & User Management)

#### 1.1 기본 인증

- ✅ **이메일/비밀번호 회원가입** (구현 완료)
  - 이메일, 비밀번호, 비밀번호 확인
  - 이메일 형식 검증
  - 비밀번호 강도 검증
- ✅ **로그인** (구현 완료)
  - 이메일/비밀번호 로그인
  - 자동 로그인 (토큰 저장)
- ✅ **로그아웃** (구현 완료)

#### 1.2 소셜 로그인 (계획)

- ⏳ **카카오 로그인**
- ⏳ **네이버 로그인**
- ⏳ **구글 로그인**
- ⏳ **페이스북 로그인**

#### 1.3 계정 관리 (계획)

- ⏳ 비밀번호 찾기/재설정
- ⏳ 이메일 인증
- ⏳ 프로필 수정
- ⏳ 회원 탈퇴

---

### 2. 캠핑장 탐색 (Campground Discovery)

#### 2.1 캠핑장 목록

- ✅ **캠핑장 리스트 조회** (구현 완료)
  - 페이지네이션
  - 썸네일 이미지
  - 기본 정보 (이름, 위치, 가격)
- ✅ **검색 기능** (구현 완료)
  - 키워드 검색
- ⏳ **필터링**
  - 지역별 필터
  - 가격대 필터
  - 시설/편의시설 필터
  - 테마별 필터 (오토캠핑, 글램핑 등)
- ⏳ **정렬**
  - 인기순
  - 가격순
  - 평점순
  - 거리순

#### 2.2 캠핑장 상세

- ⏳ **캠핑장 상세 정보**
  - 이미지 갤러리 (스와이프)
  - 기본 정보 (이름, 주소, 연락처)
  - 편의시설 아이콘
  - 운영 시간
  - 가격 정보
- ⏳ **캠핑 사이트 목록**
  - 사이트별 가격
  - 사이트별 수용 인원
  - 예약 가능 여부
- ⏳ **위치 정보**
  - 지도 표시 (네이버 맵)
  - 주소 복사
  - 길찾기 연동
- ⏳ **리뷰 섹션**
  - 평균 평점
  - 리뷰 목록
  - 사진 리뷰

#### 2.3 지도 검색

- ⏳ **지도에서 캠핑장 찾기**
  - 네이버 맵 API 연동
  - 마커로 캠핑장 표시
  - 지도 이동 시 실시간 검색
  - 클러스터링
- ⏳ **현재 위치 기반 검색**
  - 내 위치 주변 캠핑장
  - 거리 표시

---

### 3. 예약 관리 (Reservation Management)

#### 3.1 예약하기

- ⏳ **날짜 선택**
  - 캘린더 UI
  - 예약 가능한 날짜 표시
  - 체크인/체크아웃 선택
- ⏳ **사이트 선택**
  - 사이트별 가격 비교
  - 수용 인원 확인
  - 실시간 예약 가능 여부
- ⏳ **인원 선택**
  - 성인/아동 구분
- ⏳ **예약 옵션**
  - 추가 장비 대여
  - 특별 요청사항 입력
- ⏳ **예약 확인 및 결제**
  - 예약 정보 요약
  - 총 금액 계산
  - 결제 진행

#### 3.2 예약 내역

- ⏳ **내 예약 목록**
  - 예정된 예약
  - 이용 완료
  - 취소된 예약
- ⏳ **예약 상세**
  - 예약 정보 확인
  - 예약 번호
  - QR 코드 체크인
- ⏳ **예약 변경/취소**
  - 날짜 변경
  - 취소 및 환불 정책
  - 취소 처리

#### 3.3 비회원 예약 조회

- ⏳ **예약번호/이메일로 조회**
  - 비회원 예약 확인

---

### 4. 결제 (Payment)

#### 4.1 결제 수단

- ⏳ **토스 페이먼츠 연동**
  - 신용카드
  - 계좌이체
  - 간편결제 (카카오페이, 네이버페이 등)

#### 4.2 결제 프로세스

- ⏳ **결제 진행**
  - 결제 금액 확인
  - 결제 수단 선택
  - 결제 완료
- ⏳ **결제 내역**
  - 결제 상세 정보
  - 영수증 발급

#### 4.3 환불

- ⏳ **환불 요청**
  - 취소 정책에 따른 환불
  - 환불 상태 추적

---

### 5. 리뷰 및 평가 (Reviews & Ratings)

#### 5.1 리뷰 작성

- ⏳ **이용 후기 작성**
  - 별점 (1-5)
  - 텍스트 리뷰
  - 사진 업로드 (최대 5장)
- ⏳ **리뷰 수정/삭제**

#### 5.2 리뷰 조회

- ⏳ **리뷰 목록**
  - 최신순/평점순 정렬
  - 사진 리뷰 필터
- ⏳ **내가 작성한 리뷰**
  - 내 리뷰 목록

---

### 6. 마이페이지 (My Page)

#### 6.1 내 정보

- ⏳ **프로필 관리**
  - 이름, 전화번호 수정
  - 프로필 사진 변경
- ⏳ **계정 설정**
  - 비밀번호 변경
  - 알림 설정
  - 언어 설정

#### 6.2 찜 목록

- ⏳ **관심 캠핑장**
  - 찜한 캠핑장 목록
  - 찜 추가/제거

#### 6.3 고객 지원

- ⏳ **FAQ**
- ⏳ **1:1 문의**
- ⏳ **공지사항**

---

## 🎯 우선순위 (Priority)

### P0 - 필수 (MVP)

1. ✅ 회원가입/로그인/로그아웃
2. ✅ 캠핑장 목록 조회
3. ✅ 캠핑장 검색
4. ⏳ 캠핑장 상세 정보
5. ⏳ 예약하기 (날짜/사이트 선택)
6. ⏳ 결제 (토스 페이먼츠)
7. ⏳ 예약 내역 조회

### P1 - 중요

1. ⏳ 소셜 로그인 (카카오, 네이버)
2. ⏳ 지도 검색 (네이버 맵)
3. ⏳ 리뷰 조회
4. ⏳ 필터링/정렬
5. ⏳ 예약 취소
6. ⏳ 찜 기능

### P2 - 보통

1. ⏳ 리뷰 작성
2. ⏳ 프로필 수정
3. ⏳ 비회원 예약 조회
4. ⏳ 구글/페이스북 로그인
5. ⏳ QR 체크인

### P3 - 낮음

1. ⏳ 비밀번호 찾기
2. ⏳ 이메일 인증
3. ⏳ FAQ/공지사항
4. ⏳ 1:1 문의
5. ⏳ 회원 탈퇴
