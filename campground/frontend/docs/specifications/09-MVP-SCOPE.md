# MVP 범위 정의

> 최소 기능 제품 (Minimum Viable Product) 명세

## 🎯 MVP 목표

**"사용자가 캠핑장을 찾고 예약할 수 있는 최소한의 기능"**

### 핵심 가치

1. 캠핑장 정보를 쉽게 찾을 수 있다
2. 간편하게 예약할 수 있다
3. 예약 내역을 확인할 수 있다

---

## ✅ MVP에 포함되는 기능

### 1. 인증 (Authentication)

- ✅ **이메일/비밀번호 회원가입** (완료)
- ✅ **로그인** (완료)
- ✅ **로그아웃** (완료)
- ✅ **토큰 기반 인증** (완료)

### 2. 캠핑장 탐색 (Discovery)

- ✅ **캠핑장 목록 조회** (완료)
  - 페이지네이션
  - 기본 정보 (이름, 위치, 가격, 이미지)
- ✅ **검색** (완료)
  - 키워드 검색
- ⏳ **캠핑장 상세 정보** (P0)
  - 이미지 갤러리
  - 상세 설명
  - 위치 정보
  - 편의시설
  - 가격 정보
  - 리뷰 목록 (읽기만)

### 3. 예약 (Reservation)

- ⏳ **예약하기** (P0)
  - 날짜 선택 (체크인/체크아웃)
  - 사이트 선택
  - 인원 선택
  - 예약 정보 확인
- ⏳ **결제** (P0)
  - 토스 페이먼츠 연동
  - 신용카드 결제
  - 결제 완료 처리
- ⏳ **예약 내역** (P0)
  - 내 예약 목록 조회
  - 예약 상세 정보
  - 예약 상태 (예정/완료/취소)

### 4. 기본 UI/UX

- ✅ **모바일 최적화** (완료)
  - 640px 최대 너비
  - 터치 최적화 (44px 최소)
- ✅ **레이아웃 컴포넌트** (완료)
  - Header, Footer, AppContainer
- ⏳ **기본 네비게이션** (P0)
  - 하단 탭 바 (홈/예약/마이페이지)

## ❌ MVP에서 제외되는 기능

### 인증

- ❌ 소셜 로그인 (카카오, 네이버, 구글, 페이스북)
- ❌ 비밀번호 찾기/재설정
- ❌ 이메일 인증
- ❌ 프로필 수정

### 캠핑장

- ❌ 고급 필터링 (가격대, 시설, 테마)
- ❌ 정렬 (인기순, 평점순, 거리순)
- ❌ 지도 검색 (네이버 맵)
- ❌ 찜 기능

### 예약

- ❌ 예약 변경
- ❌ 예약 취소
- ❌ 비회원 예약
- ❌ QR 체크인
- ❌ 추가 옵션 (장비 대여)

### 리뷰

- ❌ 리뷰 작성
- ❌ 리뷰 수정/삭제
- ❌ 사진 리뷰

### 기타

- ❌ FAQ/공지사항
- ❌ 1:1 문의
- ❌ 포인트/쿠폰
- ❌ 알림 설정

## 🎯 MVP 완료 기준

### 기능 완성도

- [ ] 사용자가 회원가입/로그인 할 수 있다
- [ ] 캠핑장 목록을 볼 수 있다
- [ ] 캠핑장을 검색할 수 있다
- [ ] 캠핑장 상세 정보를 볼 수 있다
- [ ] 예약을 생성할 수 있다
- [ ] 결제를 완료할 수 있다
- [ ] 내 예약 내역을 볼 수 있다

### 기술 요구사항

- [ ] 모든 페이지가 모바일 최적화되어 있다 (640px)
- [ ] API 에러 처리가 적절하게 구현되어 있다
- [ ] 로딩 상태가 표시된다
- [ ] 인증이 필요한 페이지에서 리다이렉션이 작동한다
- [ ] TypeScript 타입 에러가 없다
- [ ] ESLint 경고가 없다

### UX 요구사항

- [ ] 모든 버튼/링크가 44px 이상의 터치 영역을 가진다
- [ ] 로딩 시 스켈레톤 또는 스피너가 표시된다
- [ ] 에러 발생 시 명확한 에러 메시지가 표시된다
- [ ] 뒤로가기 동작이 자연스럽다
- [ ] 폼 검증이 작동한다
