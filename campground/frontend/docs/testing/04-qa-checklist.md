```markdown
# QA 리포트 - Sprint X

**날짜**: 2024-XX-XX
**테스터**: OOO
**환경**: Production / Staging

## 테스트 결과

### 기능 테스트

- ✅ 인증: 통과 (12/12)
- ✅ 캠핑장 조회: 통과 (10/10)
- ⚠️ 예약: 부분 통과 (8/10)
  - ❌ 예약 취소 시 환불 금액 미표시
  - ❌ QR 코드 스캔 오류
- ✅ 결제: 통과 (6/6)

### 성능 테스트

- Lighthouse 점수: 92 (목표: 90+) ✅
- LCP: 2.1s ✅
- API 응답 시간: 평균 350ms ✅

### 발견된 이슈

#### 1. 예약 취소 시 환불 금액 미표시

- **심각도**: Medium
- **재현 방법**: 예약 취소 → 환불 금액 표시 안 됨
- **예상 동작**: "환불 예정 금액: XX원" 표시
- **스크린샷**: [첨부]

#### 2. QR 코드 일부 기기에서 스캔 안 됨

- **심각도**: High
- **재현 방법**: Galaxy S20에서 QR 코드 스캔 실패
- **예상 동작**: QR 코드 정상 스캔
- **영향 범위**: 체크인 불가

## 권장 사항

1. QR 코드 이슈 수정 후 재배포
2. 환불 금액 표시 기능 추가
3. 성능 최적화 지속

- [ ] 모든 테스트 통과
- [ ] 승인: ****\_\_\_\_**** (날짜: **\_\_**)
```
